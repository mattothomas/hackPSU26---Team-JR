# FounderOS

One idea. A functioning startup team in minutes.

Type a startup idea and FounderOS spins up 7 specialized AI agents — Product, Market, Business, Brand, Pitch, and Team — each analyzing a different dimension of your venture. Review each output, approve it or redirect the agent with feedback, and watch your founding team take shape.

Built for HackPSU 2026. Powered by [OpenClaw](https://github.com/openclaw/openclaw) + Gemini API.

## Setup

**1. Clone the repo**
```bash
git clone <repo-url>
cd hackpsu26
```

**2. Run the OpenClaw setup script**

This creates all config files, agent workspaces, and SOUL prompts automatically.
```bash
node openclaw-setup/setup.js
```
It will prompt for your Gemini API key, or pass it inline:
```bash
GEMINI_API_KEY=AIza... node openclaw-setup/setup.js
```

**3. Pull and start OpenClaw**
```bash
cd openclaw
export OPENCLAW_IMAGE="ghcr.io/openclaw/openclaw:latest"
docker compose pull
docker compose up -d
```

Verify it's running:
```bash
curl http://localhost:18789/healthz
```

**4. Start the backend**
```bash
cd ../backend
npm install
npm start
# Runs on http://localhost:3001
```

**5. Start the frontend**
```bash
cd ../frontend
npm install
npm start
# Opens http://localhost:3000
```

## How it works

1. You type a startup idea on the landing page and hit **Launch**
2. The backend calls each OpenClaw agent sequentially, streaming results back via SSE
3. Each agent result appears as a card in the dashboard as it finishes
4. For each section you can:
   - **Approve** — locks in that agent's output
   - **Redirect** — give the agent feedback and it rewrites its output
5. The Team panel shows the founding team as avatar cards; approve each member individually or add roles

## Agents

| Agent | Role |
|-------|------|
| `idea-parser` | Extracts problem, target users, domain, core goal |
| `product` | Defines features, user flow, tech stack, MVP scope |
| `market` | Maps competitors, market size, gap, differentiation |
| `business` | Designs pricing, revenue model, cost structure |
| `brand` | Names the startup, tagline, tone, colors |
| `pitch` | Writes the one-liner, 30s pitch, ask |
| `team-gen` | Builds the founding team with roles and week-1 tasks |
